### Initial setup:


We need to have Python 3.x (recommended 3.9 or latest) installed.
As we’re using PyQt5 framework, we need PyQt5 installed and should be in the path of the python interpreter we’re currently using
You can follow below commands


"python -m pip install PyQt5" OR incase you're using python3.x try "python3 -m pip install PyQt5"
	OR
"pip install PyQt5" OR incase you're using python3.x try "pip install PyQt5"

### Git:
If you find issues in cloning the git repo, which means you lack permissions in cloning the repository.
You need to be a collaborator in order to perform any operations on the repo. Admin or the existing members can add you to as the collaborator from the repo settings.
